#  Olympic Performance Analysis – Pakistan 

##  Project Overview
This Tableau project analyzes Pakistan’s performance in the Olympic Games and compares it with other countries.
The project is designed for students to understand sports data analysis using visual storytelling.

##  Objectives
- Analyze Pakistan’s Olympic medal performance
- Compare Pakistan with other participating countries
- Study medal trends across different Olympic years

##  Tools Used
- Tableau Desktop
- Tableau Public
- Olympic Dataset 

##  Project File
- `olympics_pakistan_analysis.twb` – Tableau workbook containing dashboards and story

##  Tableau Public Link
https://public.tableau.com/app/profile/malaika.saeed3197/viz/olympics_pakistan_analysis/ThestoryofOlympics

##  Target Audience
Students and beginners learning data visualization and Tableau storytelling.
